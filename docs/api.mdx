# API Reference Guide

This document lists all the REST API endpoints provided by the CPM platform backend, including methods, URLs, query parameters, payloads, and authorization requirements.

---

## 1. Authentication Endpoints

| Method | Endpoint | Description | Auth Level | Payload (JSON) | Response (JSON) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **POST** | `/api/v1/auth/signup` | Register a new user | Public | `{name, email, password}` | `{user: {id, name, email}}` |
| **POST** | `/api/v1/auth/login` | Login user, set session cookie | Public | `{email, password}` | `{accessToken, refreshToken, user}` |
| **POST** | `/api/v1/auth/logout` | Logout user, clear cookies | Public | None | `{status: "ok"}` |
| **POST** | `/api/v1/auth/token` | Get current access token | Authenticated | None | `{accessToken}` |
| **POST** | `/api/v1/auth/refresh`| Rotate session refresh tokens | Authenticated | None | `{accessToken, refreshToken}` |

## 5. CPM Scheduling Engine Controllers

### A. Run CPM Engine
- **Endpoint**: `/api/v1/projects/[projectId]/cpm/run`
- **Method**: `POST`
- **Role**: Project Member (Write Access)
- **Description**: Triggers topological sorting and dates computation. If the C++ process is compiled and available, it spawns `cpm_cli`. Otherwise, it falls back to the in-memory JavaScript solver.
- **Request Payload**: None
- **Response Payloads**:
  - *Status: 202 Accepted*
    ```json
    {
      "runId": "c4b4a11f-db9f-4318-8f81-a67bde56a1b2",
      "status": "processing"
    }
    ```
  - *Status: 400 Bad Request (Validation failure / Cycles)*
    ```json
    {
      "error": "CPM Engine Validation Failed: Cycle detected: task A -> task B -> task A"
    }
    ```

### B. Fetch CPM Calculation Results
- **Endpoint**: `/api/v1/projects/[projectId]/cpm/results`
- **Method**: `GET`
- **Role**: Project Member (Read Access)
- **Description**: Returns the latest calculated snapshot results containing task dates, floats, and the critical path.
- **Response Payload**:
  - *Status: 200 OK*
    ```json
    {
      "id": "c4b4a11f-db9f-4318-8f81-a67bde56a1b2",
      "projectDuration": 24.5,
      "criticalPath": ["task-1", "task-3", "task-4"],
      "details": {
        "status": "success",
        "taskDetails": {
          "task-1": { "es": 0.0, "ef": 5.0, "ls": 0.0, "lf": 5.0, "slack": 0.0, "isCritical": true },
          "task-2": { "es": 5.0, "ef": 12.0, "ls": 8.0, "lf": 15.0, "slack": 3.0, "isCritical": false },
          "task-3": { "es": 5.0, "ef": 15.0, "ls": 5.0, "lf": 15.0, "slack": 0.0, "isCritical": true },
          "task-4": { "es": 15.0, "ef": 24.5, "ls": 15.0, "lf": 24.5, "slack": 0.0, "isCritical": true }
        }
      }
    }
    ```
  - *Status: 404 Not Found* (if no scheduling run has been executed yet)
    ```json
    {
      "error": "No CPM snapshot found for this project"
    }
    ```
